===============================================
INSTRUCTIONS DE CONNEXION
===============================================

1. Lancer l'application (F5 dans Visual Studio)
2. Entrer l'un des identifiants ci-dessus
3. Entrer le mot de passe correspondant
4. Cliquer sur "Se Connecter"
5. Un message de bienvenue affichera le nom et le rôle de l'utilisateur
6. Les menus visibles dans le formulaire principal dépendent du rôle de l'utilisateur

===============================================
GESTION DES RÔLES ET PERMISSIONS
===============================================

Le système gère automatiquement les permissions selon le rôle :

- CHEF DE DÉPARTEMENT : Accès administratif complet à tous les modules
- PROFESSEUR : Accès aux modules pédagogiques (Matière, Classe, Année Académique)
- RESPONSABLE DE CLASSE : Accès aux modules de gestion de classe (Classe, Année Académique, Liste des Responsables)

Les menus non autorisés sont automatiquement masqués selon le rôle connecté.
